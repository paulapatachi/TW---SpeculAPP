Sistem de management al codului sursa

Pentru managementul codului sursa am folosit GitHub. Am pe creat pe github.com un nou repository in care am adaugat trei foldere.

Folderul "architecture" contine arhitectura aplicatiei si celelalte documente necesare etapei II.

Folderul "assets" contine resursele css si imaginile folosite in cadrul paginilor web.

Folderul "phpPages" contine paginile php care alcatuiesc site-ul web.

Am folosit GitHub pentru ca fiecare student din echipa sa aiba acces in acelasi timp la ceea ce au lucrat ceilalti colegi. Pentru a prelua modificarile colegilor faceam fetch pe GitHub si aduceam resursele pe calculatorul propriu, iar pentru a urca modificarile proprii faceam push.
